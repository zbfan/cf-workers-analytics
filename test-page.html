📦 安装指南

前置条件：

  • Node.js >= 18
  • Cloudflare 账号
  • Wrangler CLI (npm install -g wrangler)

部署步骤：

# 1. 安装依赖
npm install

# 2. 创建 D1 数据库
npx wrangler d1 create analytics-db

# 3. 更新 wrangler.toml 中的 database_id

# 4. 初始化数据库
npx wrangler d1 execute analytics-db --file=src/schema.sql

# 5. 登录 Cloudflare
npx wrangler login

# 6. 部署
npx wrangler deploy

# 7. 配置自定义域名（可选）
# 在 Cloudflare Dashboard - Workers - 触发器 中配置
